``List<T>`` é uma coleção genérica. Que funciona como um array dinâmico, ou seja, pode aumentar ou reduzir seu tamanho automaticamente conforme elementos são **adicionados** ou **removidos**.

### Principais Características
* Armazena elementos do mesmo tipo.
* Ajuste o tamanho automaticamente.
* Permite acesso ao índice.


## #Declaração e inicialização 

### Declarando vazio 
```C#
List<string> nomes = new List<string>();

List<int> numeros = new List<int>();
```

### Declarando com valores
```C#
List<string> numeros = new List<string>
{
	"Paulo",
	"Gabrielle", 
	"Arthur"
};
```

### Inicializando com capacidade (não fixa)
```C# 
List<int> numeros = new List<int>(100);
```
* Melhora o desempenho se nos soubermos quando elementos terá na lista.


## #Adicionando Elementos

*Add* - adiciona um único item
```C# 
// String
lista.Add("Paulo");

// Numero
lista.Add(15);
```

*AddRange* - adiciona múltiplos itens de outra coleção
```C# 
lista.AddRange(new int[] { 20, 30, 40 });
```

*Insert* - inseri um item em uma posição específica
```C# 
// Numero
lista.Insert(1, 15);

//string
listas.Insert(1, "Diomeneis");
```


## #Acessando Elemento 

*Acessando por índice *
```C# 
int x = lista[0];
```

*Atualizações por índice *
```C# 
lista[0] = 99;
```

*Percorrendo com for*
```C#
for (int i = 0, i< lita.Count; i++) 
{
	Console.WriteLine(lista[i]);
}
```

*Percorrendo com foreach*
```C# 
foreach (var item in lista) 
{
	Console.WriteLine(item);
}
```


## #Propriedades importantes 

**Count** - retorna quantos elementos existem.
```C# 
int quantidade = lista.Count;
```

**Capacity** - Retorna a capacidade interna (quantidade de elementos que cabem)
```C# 
int capacidade = lista.Capacity;
```


## #Remoção de Elementos

**Remove** - remove o primeiro item igual ao informado
```C#
lista.Remove(20);
```

**RemoveAt** - remove pelo índice
```C# 
lista.RemoveAt(20);
```

**RemoveRange** - remove um intervalo
```C# 
lista.RemoveRange(0, 3);
```
* remove 3 itens a partir do índice 0

**Clear** - remove tudo
```C# 
lista.Clear();
```

## Ordenação e manipulação

**Sort** - ordena a lista 
```C# 
lista.Sort();
```

**Reverse** - inverte a ordem 
```C# 
lista.Reverse();
```


## #Verificar elemento 

**Contains** - verifica se existe 
```C# 
bool existe = lista.Contains(30);
```

**IndexOF** - retorna o índice do item 
```C#
int indice = lista.IndexOf(100);
```

**Find** - retorna o primeiro que corresponde á condição 
```C# 
var resultado = lista.Find(x => x > 50);
```

**FindAll** - retorna todos que atendem á condição
```C# 
var maiorQueDez = lista.FindAll(x => x > 10);
```
