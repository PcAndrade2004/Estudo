Um método de buscar precisa retornar um valor porque o propósito da operação é entregar os dados encontrados para que outras partes do sistema possam utilizá-los.

1. **Acesso a dados** - retorna um objeto , que permite que o código chamador utilize as informações obtida. 
2. **Reutilização**  - Um método que devolve o funcionário encontrado pode ser aproveitado em diferentes operações do sistema.

## Tipos de retorno recomendados

Retorno único :
* Quando a busca deve trazer apenas um funcionário específico, o método deve retornar a classe ``funcionario`` => Caso não encontre, retorna ``null`` 

Retorno múltiplo :
* Quando poder haver vários resultados, deve retornar uma coleção, como ``List<Funcionario>`` 

## Sintaxe
```C# 
public class Funcionario
{
    public string Nome { get; set; }
    public int Idade { get; set; }
    public string Cargo { get; set; }
}

public class RepositorioFuncionario
{
    private List<Funcionario> listaDeFuncionarios = new List<Funcionario>();

    // Método que busca um funcionário pelo nome e retorna o objeto encontrado
    public Funcionario BuscarFuncionarioPorNome(string nome)
    {
        foreach (var funcionario in listaDeFuncionarios)
        {
            if (funcionario.Nome == nome)
            {
                return funcionario; // Retorna o funcionário achado
            }
        }

        return null; // Retorna null se ninguém tiver o nome informado
    }
}

```
